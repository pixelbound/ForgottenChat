ForgottenChat Version History
It aint THAT old is it?

Having a complete record of the history of ForgottenChat is simply something that HAPPENS. You dont question some things, and this is one of those things. (I do not know why exactly, but thats irrelevant). Below is a list in reverse-cronological order, the ways that ForgottenChat has Progressed since Version 1.0 Beta.

Version 2.2.1 to 2.3
  • When sending a message with Shift+Enter, the message is sent, and the edit box remains open for more input
  • Widget Scaling now works as intended
  • Modified the Chat Hook to be more compatable with other mods
  • The setting "Cascade Blacklist" has been removed and defaulted to the ON stage. As a result, if a whisper is caught by the blacklist, it will appear on your default Chat Frame unless there is another UI mod that catches it.
  • Several new key bindings to be played with
Version 2.2 to 2.2.1
  • Whisper histories are now saved per character
  • Fixed the conversation histories. The widgets were showing the FIRST 20, not the last 20 whispers.
  • Histories can now be searched for a specific word/phrase
  • Several miscellanious bug fixes
  • Updated for 1.10
Version 2.1 to 2.2
  • Added a fix for the numerous ForgottenChat .nil errors
    • If you were having issues before, I recommend deleting your old saved variables
  • The Channel frames can be scrolled to the bottom by using CTRL+MouseWheelDown
  • The /fc log page has been expanded, and conversation histories save between sessions
    • Widgets show the last 25 messages by default
    • Browse conversations by character
Version 2.0 to 2.1
  • Updated Readme file
  • Support provided for SCCN Mod
  • New option to enable ForgottenChat when talking with friends or guildies only.
  • Timestamps now appear in the Chat History.
  • ShiftClicking the Invite button will invite the participant to the users guild.
  • ShiftClicking the Ignore button will add the participant to the users friends list
  • AFK and DND message report to the ForgottenChat widgets.
  • Added a fix for Chatbox
  • Fixed an error that occured when the user disabled ForgottenChat
  • Fixed Channel Support Combat Minimize/Maximize
  • The Channel Support can now be toggled
Version 1.4 to 2.0
  • Fixed a bug which will allow alphanumeric hotkeys to be used
  • Fixed a bug which caused a stack overflow error when clicking on a Paperdoll Item after disabling and re-enabling FC.
  • Items can now be linked from AllInOneInventory.
  • Added an option to pass Blacklisted whispers to the default blizzard chat frames.
  • Shift clicking on the Alias bar of the widget will perform a who of the specified person
  • A profile manager has been included to copy and paste profiles between characters.
  • Fixed a bug that prevented items being un-linkable when the widget is first opened
  • Channel Support!
  • 6 new Whisper Widgets to play with.
Version 1.3 to 1.4
  • Audio alert when new message is diverted to a minimized widget.
  • Items can now be linked from the paper doll and the Bank in addition to inventory.
  • Fixed a bug that caused multiple whispers to appear.
  • When minimized widgets receive whispers, the number of Unread whispers is displayed beside the name on the widget.
  • Errors should no longer be thrown when starting a new conversation when in Battlegrounds.
  • Added a reply keybinding that brings up the edit box on the most recent conversation.
Version 1.2 to 1.3
  • Retrieving character info has been streamlined slightly. ForgottenChat maintains info from old conversations in addition to scanning the current /who for a match before performing its own search.
  • A Blacklist of phrases has been included which will automatically ignore messages that Start with them. "[CTA]" will block all CallToArms messages. "<GA" will block all GuildAds messages "fort " will block Whispercast Fortitude requests... however it may cause false positives
  • Item links can now be put into the edit boxes and sent in a whisper.
  • Timestamps are now in 24 hour time.
  • Slash commands are now passable in any of the
  • Widget EditBoxes.
  • Minimum Widget size has been removed
  • Now able to hide the Action Buttons
  • Who information retrieval can be disabled
Version 1.1 to 1.2
  • Enabled Timestamps
  • Expanded on the Conversation History
  • Text size can now be changed.
  • Added an 'Ignore' button to each widget
  • Changed the look of the EditBoxes
  • Provided support for non-english clients
  • Added /fc <name> to open new widgets
  • Provided a button to reset widget positions.
  • Included a readme.txt
  • Added /fc enable and /fc disable commands
  • Ignores messages with both < and > in them. This prevents the showing of certain 'hidden' whispers. Special considerations made for CallToArms and GuildAds whispers.
Version 1.0 to 1.1
  • Replies and /r work as intended now
  • Added an 'Invite' and 'Inspect' button on the widgets
  • Fixed a bug with loosing EditBox focus randomly
