ForgottenChat User Profiles
What is your is yours, and mine is mine.

Since basic Settings are saved on a Per Character, Per Server basis the ability to copy from one character to another required manipulation of the SavedVariables.lua file. In version 2.0, the ability to copy, paste, and delete profiles has been added to add a degree of uniformity.

A Profile consists of the Widget Behavior and Widget Appearance settings. It does NOT include Alias Lists, Blacklist entries and Channel Settings. See the Settings page for how these are saved and what they do.

Copying a Profile:
Copying a profile can be done simply by selecting the profile from the list on the ForgottenChat configuration frame and pressing the Copy button. Once a profile is copied, further actions can be taken. the last profile copied will remain copied until log out.

Pasting a Profile:
Once a profile has been copied, it can be pasted over a currently existing one. At the moment, you cannot create a new profile off of a copy, you can merely rewrite ones that exist. Simply select the profile you wish to paste over and click the paste button.

Deleting a Profile:
You can delete a profile that you no longer use simply by selecting the profile and pressing the delete button. You cannot retrieve a previously deleted profile. You cannot delete the profile you are currently logged into.

Exploding a Profile:
Once you have copied a profile, you can 'explode' it over all other profiles saved in memory. This permanently overwrites all old settings.
