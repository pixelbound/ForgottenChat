I believe Tabbed Widgets would make whispering easier. Are there any plans to implement tabs with ForgottenChat?
While the suggestion of tabs is not a new one, the Dev Team will not be implementing tabs in the forseeable future. ForgottenChat was designed to consist of multiple and individual whisper widgets, and having tabs will obviously break this concept. Neither will it be an in game option due to the amount of coding required for something the developer has absolutly no desire to use.

ForgottenChat closes standard World of Warcraft frames when a new conversation is started. How do I fix this?
The closing of the Auction House, Mailbox, and other frames when a new message is received is an unfortunate side-effect caused by ForgottenChat doing a hidden /who of the person talking to you. FC does this to fill in the /who field at the bottom of each widget with the appropriate information. By setting the Ignore Who option. The performing of an automatic /who is streamlined as much as possible to prevent such an occurance but it still happens.

ForgottenChat is causing an error, what do I do?
If ForgottenChat starts to complain and throw off errors, there are very few things that can be done by the average user to solve the problem. An error message is best dealt with by the Dev Team, so follow the instructions on Bug Reporting. Make sure to follow the instructions as closely as possible since information is the most valuable tool in fixing these issues.

I cannot link from mod X, can you fix it?
As much as the Dev Team would love to include support for other addons within ForgottenChat, it simply is too great an undertaking for too little return. There are literally hundreds of UI mods that have special considerations regarding linking, and it is simply not worth the effort to manage them all. Each UI mod fixed in this way requires a considerable amount of code and research which when compared to the number of users using Mod X and ForgottenChat is severly unbalanced.

Some of the more popular UI mods are taken in for consideration such as AllInOneInventory since it has more than 400,000 downloads as of February 5th, 2006.

If you have an understanding of LUA programming, check out the tutorials page for instructions for setting up your own ItemLinks from other mods.

Where can I download the latest version of ForgottenChat?
ForgottenChat is hosted by Curse-Gaming, WorldOfWar, and WoWInterface. The latest version of ForgottenChat can be found at any one of these sites.

ForgottenChat keeps picking up messages from other UI mods, can I hide these?
Mods such as CT_Raid use whispers to communicate between players and due to the method of catching whispers in ForgottenChat, they do not remain hidden. The Blacklist is the method of hiding whispers that follow a predictable pattern. CallToArms affixes the "[CTA]" tag in front of all their whispers, and GuildAds uses "<GA". By placing these entries in the blacklist, ForgottenChat will automatically ignore them.

ForgottenChat does not report the location along with the other /who information.
Since ForgottenChat performs a single /who for every person talked to, it records all the information for later use to prevent un-necessary /who operations. In doing so, player location would become quickly out of date and need refreshed causing more traffic between the server and client as well as causing miscellanious WoW frames to close. It was decided that location is not important enough to warrant the extra time requirement and was replaced by ShiftClicking the widget titlebar.
