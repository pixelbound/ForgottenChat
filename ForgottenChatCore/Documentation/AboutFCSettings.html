ForgottenChat Configuration Details
You want it to do WHAT?

All settings for ForgottenChat can be changed through the ForgottenChat Options menu (accessable by the /fc slash command) The manner in which these settings are saved and their definitions are explained here.

Behavior Settings
These settings depict the way that the whisper widgets behave during run time. They are saved on a Per-Character Per-Server basis. "Euler of Kel'Thuzad" has a differant profile than "Euler of Aegwynn" A description of individual Behavior Settings are explained below.

Load Minimized
Default: OFF
When this is enabled, newly created widgets will start up minimized
Enable Flashing Default: ON
When a new message is received in a minimized widget, the title of the widget changes to the color defined by the 'Flashing' Effect color. No actually flashing occurs.
Cycle Widgets Default: ON
When all widgets are in use and a new conversation is initated, ForgottenChat will automatically close the oldest conversation to make room. If this is disabled, no new widget will be created, you will be allerted you missed the message, it will be placed in the ForgottenChat Log, and your Busy Message is sent.
Use Aliases Default: ON
Replaces the name of the person you are whispering with a name of your own choosing.
Combat Hide Default: OFF
When entering combat, all widgets are hidden from view. When exiting combat, they all reappear.
Combat Minimize Default: ON
When entering combat, all unminimzed widgets are minimized. Upon exiting combat, only those that were automatically minimized are re-maximized.
Click to Interact Default: ON
Enabling this will show the widget's edit box when it is clicked as opposed to showing on mouse over.
Use Mouse Wheel Default: OFF
Removes the scroll buttons on the widget and are replaced by the mouse's scroll wheel.
Maximize Up Default: ON
All widgets maximize up and minimize down instead of maximizing down and minimizing up.
Show Timestamp Default: ON
Enables the time stamps on messages received when this option is set.
Play Alert: Default: ON
When a minimized widget receives a new whisper, an audible alert is sounded. Default sounds must be appropriatly configured.
Timeout Default: 120
This is the time in seconds after the last message is received from a widget before it is marked as inactive and is automatically closed. A check for inactive widgets is performed when a new target is selected and marked
Max Widgets Default: 6
Sets the maximum number of widgets allowable on screen at any given time. 0 will prevent any incoming messages from being displayed. Maximum setting is 12.
Busy Message Default:
This is the message that is sent when all widgets are in use.
Selective Widgets Default: OFF
Enabling this option will only open a ForgottenChat widget for people within your guild, on your friends list, or on your alias list. Any other conversations will be directed to the default WoW Chat Frames.
Save History Default: OFF
When this option is disabled, any old conversations that were in the SavedVariables will automatically be deleted upon login. If it is enabled, any and all old whisper conversations will remain in the Saved Variables indefinatly. NOTE: Leaving this enabled for a long period of time could use up valuble memory space!

 

Appearance Settings
These settings depict the way that the whisper widgets appear during run time. They are saved on a Per-Character Per-Server basis. "Euler of Kel'Thuzad" has a differant profile than "Euler of Aegwynn" A description of individual Appearance Settings are explained below.

Widget Alpha Level
Default: 90%
This is the transparency level of all the whisper widgets' and channel widgets' background. Values between 0 and 100 (fully transparent to opaque respectivly)
Height/Width Default: 150/400
The height and width of all whisper widgets.
Text Size Default: 14
The size of the text in both the whisper and channel widgets.
Hide Buttons Default: OFF
Hides the target, trade, invite, inspect and ignore buttons on the whisper widgets
Ignore Who Default: OFF
Enabling this will prevent ForgottenChat from forming a /who to determine race, class, level and guild to display on the bottom of the widget. Enable this if you are having issues with your own /whos. When an automatic /who is performed, the results are saved for the session to prevent repetative requests. Having this enabled will also prevent ForgottenChat from closing certain frames when a new conversation is started. (such as the Auction House or Mailbox)
Count Unread Default: ON
Adds a number beside the name or alias showing the number of unread messages that widget contains. Only appears on minimized widgets by definition.
Widget Scale Default: 100%
Use this slider to change the scale of the widget. Maxed at 200%, 25% minimum. Doubles, and quarters the size of the widget respectivly.

 

Aliases Settings
Aliases are nicknames that are associated with specific characters. If a conversation is started with someone on your alias list, their character name on the top of their widget will be replaced by their alias if Use Aliases is enabled.

These settings are saved on a Per-Server basis. "Euler of Kel'Thuzad" has a differant alias list than "Euler of Aegwynn", but the same as "Bovi of Kel'Thuzad"

 

Blacklist Settings
Before a whisper is sent to it's appropriate widget, it is scanned by all the entries in the blacklist. If the whisper starts with any of the blacklisted phrases, it is ignored by ForgottenChat. This feature was included to bypass automated whispers from other UI mods in the World of Warcraft. Most of them start with a recognizable phrase. This is not all inclusive, but provides a modicum of protection.

These settings are saved on a general basis. "Euler of Kel'Thuzad" has the same blacklist entry as "Bovi of Aegwynn"

 

Profile Settings
A profile is the collection of Appearance and Behavior settings associated with a specific character. Check here for specifics.

 

Channel Settings
These settings record ForgottenChat channel widget behavior and to an extend, the appearance. They are stored on a Per-Character Per-Server basis. "Euler of Kel'Thuzad" has a differant setting than "Euler of Aegwynn" A description of individual Channel Settings are explained below. Each Channel widget can have distinct settings.

Widget Title
 
This is what you wish to call the widget.
Combat Minimize  
Automatically minimizes the widget when entering combat.
Combat Maximize  
Automatically maximizes the widget when entering combat
Use This Widget  
Simply shows or hides this widget.
Disable Channel Support  
This feature was included to fix the compatability issues with other Chat based UI mods. This toggle has not been fully tested as of the latest release. After changing this feature, a /console reloadui may be required.
