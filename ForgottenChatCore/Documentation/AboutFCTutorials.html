Setting up Channel Support for /guild and /party

I have been asked a couple times how to set up the guild channel and party raid channels within their own widget. This can currently be done with the Channel Support feature for any channel you chose, so this tutorial can be expanded beyond /g and /p. I will warn you that a thorough understanding of the Default WoW chatframes will make this much easier, so I shall provide a rundown.

There are seven (7) standard WoW chatframes. New users or characters will only be shown two of them, one with all the default stuff shown on them, such as loot, /1-/4, whispers, /p etc, and another for the Combat Log. These are numbered (respectivly) #1, and #2. Users can create more chatframes (#3-#7) and assign any combination of channels to any combinations of chatframes.

For the sake of argument, pretend I am using this basic setup:

  1. All Channels other than Combat Logs!
  2. Combat Log
  3. Unused
  4. Unused
  5. Unused
  6. Unused
  7. Unused

Disable Anonymous Statistics Reporting for Icecrown characters.

Your first question will be "huh WHAT? what is this?" and I shall tell you. Since my main character is on the realm Icecrown, I thought it would be interesting to know how many people use ForgottenChat, and what sort of pattern there is among my users. To that end, I created the Icecrown Anonymous Report system. that means that if you are logged into Icecrown and have ForgottenChat enabled and my main (Rachelle - Alliance) is logged in, you will send me an automated whisper about yourself. The info that I get is the following:

  • Your characters name
  • Your guild name
  • Your current level
  • Your class
  • Your race
  • Your rank within your guild
  • The version of ForgottenChat you are currently using

For instance, my main 'Rachelle' sends to herself upon login this whisper:
[Rachelle] <FC> Using Version: 2.2.1 Level: 60 Class: Mage Race: 'Human' Guild: Knights Who Say Ni' GRank: 'Kniggit'

Forgotten Chat hides this from the user, and I have another mod on my end which tracks all these whispers for statistics purposes. This was designed to be as unintrusive as possible, but I can understand some people not wishing me to have this information, so here is how to disable it:

Open up the ForgottenChat.lua file in your favorite text editor, and search for the line of code starting with this text:
SendChatMessage("<FC> Using Version:
Either delete the entire line, or comment it out by placing two dashes in front of it:
--SendChatMessage("<FC> Using Version:

If more code is deleted or commented out, I cannot guarentee that ForgottenChat will still function properly. Every time you update ForgottenChat, you must repeat this method. This is only if you play regularly on the Alliance side of Icecrown, otherwise nothing will happen, nobody will get this information.
